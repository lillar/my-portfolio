# my-portfolio

A personal portfolio showcasing data visualization projects built with D3 and React.

## Pages

- **D3 Course** — charts built while following the [React Graph Gallery](https://www.react-graph-gallery.com/) D3 course
- **#30DayChartChallenge** — one chart per day for 30 days, following the [Edition 2026](https://github.com/30DayChartChallenge/Edition2026) prompts

## Tech

- React + Vite
- React Router
- D3 (scales and layouts only, React JSX for rendering)

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Live at: `https://lillar.github.io/my-portfolio/`
